# Example Documents

This directory contains example documents for testing the demo.

## sample_docs.txt

Contains three example documents about famous landmarks:
- The Eiffel Tower
- The Statue of Liberty  
- The Great Wall of China

You can copy and paste these into the demo to test different questions like:
- "Which monument was a gift from France?"
- "What is the tallest structure mentioned?"
- "When was the Statue of Liberty dedicated?"

## Usage

1. Open `sample_docs.txt`
2. Copy the entire contents
3. Paste into the "Documents" textarea in the demo
4. Ask a question about the content
5. Compare results between Normal RAG and CLaRa modes
